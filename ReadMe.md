# Fusion SRE

to run application locally:

export AWS credentials in terminal

```angular2html
export AWS_ACCESS_KEY_ID=your-access_key
export AWS_SECRET_ACCESS_KEY=your_secret_key
export AWS_DEFAULT_REGION=your_region
```
### Run terraform
``
terraform init
``

``
terraform apply
``

### Run App
``sbt run``
